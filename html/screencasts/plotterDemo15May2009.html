Danse Plotter is a general 2-D/3-D plotting package which

  • uses the advanced Math Types scientific API
  • supports text, open netcdf, and hdf5 binary data
  • dynamically remapa multidimensional data to new axes, animate
  • supports interactive manipulation of data
  • can create plot arrays, overlay plots, compare experiment vs. theory
  • allows collaboration â€“ two people control the same plot at the same time
  • supports python scripting

To access a tutorial on text-formatted plotting, see the wiki. Test data sets include:

  • Line plot
  • 2D surface
  • 3D volumetric data

Press "Launch" to activate Danse Plotter in Java Web Start mode.

Note: you must have a Java Virtual Machine (JVM) installed. If you do not, clicking "Launch" will likely cause your browser to alert you to this and install one automatically. It is highly recommended to use the JVM from Sun Microsystems. Using the OpenJDK (default on Ubuntu OS) does not work due to bugs in their webstart launcher.
